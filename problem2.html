Problem 2

"By considering the terms in the Fibonacci sequence whose values do not exceed four million, find the sum of the even-valued terms."

Fn = Fn-1 + Fn-2

The seed numbers for the equation are either 0, 1 or 1, 1; which one I start with will not make any difference to the answer.

function init()
{
  var total = 0, a = 1, b = 1, swap;
  var output = document.getElementById("output");
  for (var i = 0 ; i < 40000 ; i++)
  {
    a += b ;
    if(a > 4000000) { break; }
    else if(a % 2 === 0) { total += a; }
    else {}
    swap = a; a = b; b = swap;
  }
  output.innerHTML += total;
}
document.addEventListener("DOMContentLoaded", init, false);

This solution works as follows. I have defined four variables: total, A, B and swap. Total is used to store the sum of even Fibonacci numbers; A represents Fn-2 and B represents Fn-1; finally swap is used to store the contents of the A variable while the values of A and B are being swapped.

A and B have the seed values of 1 and 1. This algorithm is looped and the loop starts here. The value of A is made equal to the sum of A and B (eg so taking the seed values A = 1 + 1 = 2). If A is greater than 4,000,000 the loop stops; otherwise it checks if a is an even number (if the remainder of dividing A by 2 = 0, then A is even). If A is even then total = total + A, otherwise the value of total is not updated. Once these checks are complete the contents of variables A and B are swapped around. The loop ends if A > 4,000,000 otherwise it starts again.
