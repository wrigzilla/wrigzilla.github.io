Problem 1

"Find the sum of all the multiples of 3 or 5 below 1000."

function init()
{
  var total = 0;
  for (var i = 1; i < 1000; i++)
  {
    if(i % 3 === 0 || i % 5 === 0) { total += i; }
  }
  document.getElementById("output").innerHTML += "<br>" + total;
}
document.addEventListener("DOMContentLoaded", init, false);

My solution works by having a loop with a counter (the variable i) which starts at 1 and ends at 999. I starts at 1 and every iteration checks to see if 3 or 5 is a factor. If it is a factor the variable total is made equal to total plus i. After all numbers greater than 0 and less than 1000 have been checked the variable total is printed.
